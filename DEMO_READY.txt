================================================================================
                        EMERGENCY TROUBLESHOOTING
================================================================================

PROBLEM: Can't login
FIX: Check Supabase connection, verify credentials, check browser console

PROBLEM: Blank dashboards
FIX: Refresh page, clear cache, check if demo data is seeded

PROBLEM: OCR very slow (>10 seconds)
FIX: NORMAL - Explain timeout feature exists, click "Continue" button

PROBLEM: Upload fails
FIX: Try smaller PDF file, check file format, verify storage

PROBLEM: Wrong role dashboard appears
FIX: Logout completely, sign in again with correct account

PROBLEM: Page won't load
FIX: Check internet, try incognito mode, clear cookies

PROBLEM: Network error
FIX: Check Supabase connection, try again, have screenshots backup
